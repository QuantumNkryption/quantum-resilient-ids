### **Quantum-Resilient IDS Simulation Framework**

This repository contains the full Python implementation of the Quantum-Resilient Intrusion Detection System (IDS) simulation framework used in my MSc thesis. It includes synthetic traffic generation, PQC-distorted traffic simulation, feature preprocessing, model training, and evaluation.

Contents

\- generate\_baseline\_traffic.py: Synthetic baseline traffic generation

\- pqc\_transform.py: PQC handshake inflation and timing delay simulation

\- preprocess\_features.py: Feature scaling and encoding

\- train\_model.py: Neural network model training

\- evaluate\_model.py: Model evaluation metrics

\- requirements.txt: Python dependencies



Usage

\- Clone the repository: git clone https://github.com/yourusername/quantum-resilient-ids.git

\- Install dependencies: pip install -r requirements.txt

\- Run the main script: python train\_model.py
